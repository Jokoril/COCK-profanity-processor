# Compliant Online Chat Kit Quick Start Guide

## Step 1: Run as Administrator

Right-click on **CompliantOnlineChatKit.exe** and select **"Run as administrator"**
<div align="center">
    <img src="https://i.ibb.co/mrk9Q9wb/Runasadmin.png" height="350" alt="admin">

<br/>

**Note:** Administrator privileges are required for the app to function properly for most games.
</div>

---

## Step 2: Set Your Hotkeys

Configure your preferred hotkeys in the settings window:

- **Normal Optimize Hotkey:** F12 (default)
- **Force Optimize Hotkey:** Ctrl+F12 (default)
- **Toggle Hotkeys On/Off:** Ctrl+Alt+C (default)

<div align="center">
    <img src="https://i.ibb.co/cSFVhBhJ/Recordhotkey.png" height="350" alt="hotkey">

<br/>

**Tip:** Click "Record..." to set a custom hotkey combination.
</div>

---

## Step 3: Test Your Message

Type your message and **press the HOTKEY** to send it.
<div align="center">
    <img src="https://i.ibb.co/67yt7DbD/Screenshot-2025-12-28-082901.png" height="150" alt="hotkey">

<br/>

**Important:** DO NOT click the Send button - Use your hotkey instead!
</div>
